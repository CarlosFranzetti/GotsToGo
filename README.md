# GotsToGo

**Find accessible bathrooms in New York City, stress-free.**

Search clean, accessible restrooms across all five boroughs with real-time info and honest reviews.

## Features

- 🔍 Search accessible restrooms across NYC's five boroughs
- ⭐ Real-time info and honest reviews
- ♿ Accessibility-first design
- 📍 Five borough coverage

## Tech Stack

- TypeScript
